Form Generetor by 'two clicks'

This application generates HTML and typescript form-code for angular app, based on JSON string that you`ve passed.

  • Github
  • Pateron

How it works

Idea is to simplify form buildng proccess by auto-generating form-code that you than put in your application and magicaly it works.

How to use

There are two ways to generate form.

  • Generate form based on JSON string
  • Interactive way (coming soon)

When to use

Although you can use it to generate small and simple forms, main idea for this tool is to be used when there is a need for big, complex form with lot of fields.
