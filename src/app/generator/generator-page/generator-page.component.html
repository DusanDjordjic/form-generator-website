How it works

There are two ways to generating form structure. JSON way and interactive way

JSON way

You can generate form html and typescript code from JSON string. There are Controls representing input field and Groups that are just collections of Controls.

Groups

Each form structure must start with the group. First group is form tag and it's must have. Group is nothing but an object that has name and children properties. Name property is of type string and it represents the name of the group and children is an array of controls or other groups. You see now how you can nest one in another.

Controls

For the form to work we must add some controls, what`s the use of the form if we dont have any fields to write to.

Control is an object that has nameproperty, same as group and type property that defines type of an input. Types are the same as html input types.
